## 목표
- 엑셀파일을 업로드 하면 환율 계산후 물가 비교후 엑셀파일로 변환해주는 Python Typer 기반 CUI Application

## 사용방법
1. 엑셀에 데이터 입력 (물품명, 한화 가격, 엔화 가격)
2. `python main.py {엑셀 파일 이름}`
3. 엑셀 파일의 환율, 결과 시트 확인
