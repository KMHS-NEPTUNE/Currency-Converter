# 사용방법

## 환율 계산
1. 엑셀에 데이터 입력
   - format.xlsx 파일을 이용해 값입력 (example.xlsx 참고)
![img.png](img.png)

2. hwanyulgyesangi 실행
   - `python main.py {엑셀 파일 이름}`
   <br />또는
   - [실행파일 다운로드](https://github.com/wegotoajapan/hwanyulgyesangi/releases/latest) 후 `.\hwanyulgyesangi.exe {엑셀 파일 이름}`

3. 완료!
